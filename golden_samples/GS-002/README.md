# GS-002: 简单桁架结构静力学分析

## 案例概述

GS-002 是一个简单的静定桁架结构案例，用于验证桁架单元（T3D2）的解析和静力学计算。

## 结构参数

### 几何构型

```
         Node 3 (apex, P = 1000 N downward)
        / \
       /   \
      /     \
Node 1       Node 2
(fixed)     (roller_y)
```

- **跨度**: L = 10 m
- **高度**: h = 8.66025404 m (等边三角形)
- **杆件数**: 3
- **节点数**: 3

### 节点坐标

| 节点 | X (m) | Y (m) | Z (m) | 边界条件 |
|------|-------|-------|-------|----------|
| 1 | 0.0 | 0.0 | 0.0 | 固定端 |
| 2 | 10.0 | 0.0 | 0.0 | Y向滚动 |
| 3 | 5.0 | 8.66025404 | 0.0 | 自由端 |

### 杆件属性

| 杆件 | 连接 | 长度 (m) | 面积 (m²) | 角度 |
|------|------|----------|-----------|------|
| 1 | 1 → 3 | 10.0 | 10.0 | 60° |
| 2 | 2 → 3 | 10.0 | 10.0 | 120° |
| 3 | 1 → 2 | 10.0 | 10.0 | 0° |

### 材料参数

- **材料**: STEEL
- **弹性模量**: E = 210 GPa
- **泊松比**: ν = 0.0 (桁架单元无横向变形)

### 载荷条件

- **类型**: 集中力
- **位置**: 节点3
- **方向**: Y方向（垂直向下）
- **大小**: P = -1000 N

## 理论解

### 支座反力

根据静力平衡方程：

```
ΣFy = 0:  R1y + R2y = P = 1000 N
ΣM about node 1 = 0:  R2y × 10 = P × 5

→ R1y = R2y = 500 N (向上)
ΣFx = 0:  R1x = 0 (节点1固定端)
```

### 杆件轴力（节点法）

**节点3分析**:
```
ΣFy = 0:  -F1×sin(60°) - F2×sin(60°) + 1000 = 0
         (斜杆的Y分量向下，载荷向上为正)

由于对称: F1 = F2 = F

2 × F × sin(60°) = 1000
F = 1000 / (2 × 0.866) = 577.35 N (压力)

ΣFx = 0:  F1×cos(60°) - F2×cos(60°) = 0 ✓
```

**杆件1和杆件2**: 压力 F = -577.35 N

**节点1分析**:
```
ΣFx = 0:  F1×cos(60°) + F3 = 0
         -577.35 × 0.5 + F3 = 0
         F3 = +288.68 N (拉力)
```

### 轴力汇总

| 杆件 | 轴力 (N) | 类型 |
|------|----------|------|
| 1 | -577.35 | 压力 |
| 2 | -577.35 | 压力 |
| 3 | +288.68 | 拉力 |

### 应力计算

```
σ = F / A = 577.35 / 10 = 57.735 MPa
```

### 节点位移

由于杆件轴向变形：

```
δ = F × L / (E × A)
  = 577.35 × 10 / (210×10³ × 10)
  = 2.75 × 10⁻⁶ m
```

但使用单位载荷法（能量法）计算节点3的垂直位移：

```
δ_v = Σ(F_i × F_unit_i × L_i) / (E × A_i)

对于P=1000N作用下的内力 F_i = -577.35 N
对于单位载荷(1N)作用下的内力 F_unit_i = 0.577

δ_v = 2 × (-577.35 × 0.577 × 10) / (210×10³ × 10)
    = -2.37 × 10⁻⁹ m
```

**注意**: 由于截面面积很大（10 m²），位移非常小。
