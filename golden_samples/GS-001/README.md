## 案例概述

这是一个用于验证AI-Structure-FEA系统的基准案例，采用**悬臂梁**模型。

⚠️ **重要说明**: 原 `expected_results.json` 和本文档曾错误描述为"简支梁"，现已更正为实际案例类型——悬臂梁。

## 模型参数

### 几何参数

- **梁类型**: 悬臂梁 (Cantilever Beam)
- **长度**: L = 100 m
- **截面**: 矩形 b × h = 10 m × 10 m
- **横截面积**: A = 100 m²
- **惯性矩**: I = bh³/12 = 833.33 m⁴

### 材料参数

- **材料**: STEEL (钢材)
- **弹性模量**: E = 210 GPa
- **泊松比**: ν = 0.3
- **剪切模量**: G = 80.77 GPa

### 边界条件

- **固定端**: 左端 (节点 1, 12, 23, 34) 全约束 (UX=UY=UZ=0)
- **自由端**: 右端 (节点 11, 22, 33, 44) 无约束

### 载荷条件

- **载荷类型**: 集中力 (CLOAD)
- **作用节点**: 11, 22, 33, 44 (自由端)
- **每个节点**: -100 N (Y方向)
- **总载荷**: P = -400 N

### 网格参数

- **单元类型**: C3D8 (8节点六面体单元)
- **单元数量**: 10
- **节点数量**: 44
- **网格来源**: `calculix_cases/cantilever_beam/cantilever_beam.inp`

## 理论解

### 欧拉-伯努利梁理论

对于悬臂梁（固定端受力），理论公式：

#### 最大弯矩（固定端）

```
M_max = P × L = 400 × 100 = 40,000 N·m
```

#### 最大弯曲应力（固定端最外层纤维）

```
σ_max = M_max × c / I = P × L × (h/2) / I

其中 c = h/2 = 5 m (中性轴到边缘距离)

σ_max = 40,000 × 5 / 833.33 = 0.00024 MPa = 240 Pa
```

#### 最大挠度（自由端）

**弯曲变形**:
```
δ_b = P × L³ / (3 × E × I)
    = 400 × 100³ / (3 × 210×10⁹ × 833.33)
    = 0.000000762 m = 0.00076 mm
```

**剪切变形** (Timoshenko修正):
```
δ_s = P × L / (k × A × G)
    = 400 × 100 / (0.833 × 100 × 80.77×10⁹)
    = 0.000000006 m = 0.000006 mm
```

**总挠度**:
```
δ_max = δ_b + δ_s ≈ 0.000768 mm
```

#### 剪切/弯曲变形比

```
δ_s / δ_b = 0.78%
```

## FEA结果（基准值）

> ⚠️ 由于3D实体单元与理想梁理论假设差异巨大，**以FEA结果作为本案例的验证基准**。

### 位移结果

| 节点 | 位置 | UX (m) | UY (m) | 说明 |
|------|------|--------|--------|------|
| 11 | 自由端角点 (100, 0, 0) | -0.03687 | -0.49356 | 最大位移 |
| 22 | 自由端 (100, 10, 0) | -0.03687 | -0.49356 | - |
| 33 | 自由端 (100, 0, 10) | -0.03687 | -0.49356 | - |
| 44 | 自由端角点 (100, 10, 10) | +0.03687 | -0.49356 | - |
| 1 | 固定端角点 (0, 0, 0) | 0 | 0 | 约束节点 |

### 应力结果

| 节点 | 位置 | SXX (MPa) | 说明 |
|------|------|-----------|------|
| 1 | 固定端角点 | -190.08 | 最大压应力 |
| 12 | 固定端 | +190.08 | 最大拉应力 |

### 关键发现

1. **UY位移** (493.56 mm) 远大于理论预测 (0.00076 mm)
2. **UX位移** (36.87 mm) 显著存在，理论预测为0
3. **应力水平** (190 MPa) 与理论 (0.00024 MPa) 差异巨大

### 差异原因分析

| 因素 | 影响说明 |
|------|----------|
| 3D实体 vs 梁理论 | C3D8单元不是基于梁理论，行为完全不同 |
| 载荷分布 | 4节点分布载荷 vs 理想点载荷 |
| 圣维南效应 | 10单元网格稀疏，边界效应显著 |
| 深梁效应 | L/h=10，剪切变形和泊松效应显著 |
| 泊松收缩 | 横向收缩导致UX位移 |

## 注意事项

1. ⚠️ **本案例理论与FEA存在巨大差异**，这是正常的，因为：
   - FEA使用3D实体单元，不适用欧拉-伯努利梁理论
   - 网格稀疏导致圣维南效应显著
   - 深梁(L/h=10)的剪切变形不可忽略

2. 本案例主要用于验证：
   - FRD文件解析器的正确性
   - FEA结果的完整性
   - 系统的数据提取能力

3. 如需验证梁理论，应使用更细密的网格和更高的L/h比
