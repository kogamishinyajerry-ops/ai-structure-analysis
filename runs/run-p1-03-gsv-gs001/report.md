## 3. Root-cause analysis — why C3D8 underpredicted

The legacy C3D8 (fully-integrated trilinear hex) element suffers from
**shear locking** in pure bending when only one element is used through
the beam thickness. Parasitic shear strains inflate the element stiffness,
so the predicted tip deflection (0.4936 mm, -35.2% vs theory) and the
fixed-end bending stress (190.1 MPa, -20.8% vs theory) are both
systematically low.

**Fix applied (`gs001.inp`):** switch element type to `C3D8I` —
incompatible-modes trilinear hex — which adds internal deformation
modes that cure the shear-locking pathology without changing the mesh
node layout. Result: tip deflection 0.7567 mm (-0.68%), top fibre
SXX +248.5 MPa (+3.54%). Both metrics under the 5% threshold.

**Regression protection:** `gs001_c3d8_shearlock.inp` is preserved as a
canonical failure fixture. `tests/test_golden_sample_validation.py`
asserts the legacy deck must fail the validator — if it ever starts
passing, either the validator tolerance has been relaxed silently or
the reference values were miswritten.
